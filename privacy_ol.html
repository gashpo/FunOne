隱私權政策

我們致力於保護您的隱私

了解有關 FunOne 如何收集和使用資料以及您作為 FunOne 用戶的權利的更多資訊

2025 年 2 月更新
  1. 適用範圍

    本政策適用於您使用本服務時，涉及您個人資料的蒐集、處理及利用。但不包括透過本服務連結至第三方經營的應用程式或網站。

  2. 資料的蒐集目的與類別

    本公司蒐集您的個人資料，以提供您完整的會員服務，包括但不限於會員管理、行銷推廣、客戶服務、交易處理、數據統計分析等。我們可能蒐集以下類別的個人資料：

    1. 您提供的基本資料及特定資訊，如姓名、國籍、電子郵件、聯絡方式、語言設定、會員暱稱及客服意見等。

    2. 裝置及連線資訊，如裝置型號、作業系統、網際網路服務提供者、行動裝置設定、IP 位址、瀏覽器類型及其他行動裝置所獨有而可供辨識之資訊等。

    3. 我們可能必須將您的相關資訊提供給第三方的服務提供者，以便處理您的交易或為您提供服務。有關於這部分個人資料的使用將遵守服務提供者的隱私權政策。

    4. 與您的購買相關之資訊。您付費購買本服務時，我們可能會蒐集購買日期、數量、項目、信用卡號碼等交易資訊。

    5. 您與第三方服務之相關資訊。在您將本服務與第三方服務連結時，我們可能蒐集您在第三方服務的相關資訊，例如登入認證資料、個人檔案等。當本服務與第三方服務進行聯動時，我們會基於識別及紀錄之目的將您部分的資訊提供給合作第三方。

    6. 我們會記錄您使用本服務的足跡，以提供您更佳的個人化服務，方便您參與個人化的互動活動。例如，我們會運用 Cookie或其他類似技術之工具如localStorage、sessionStorage等 (下統稱”Cookie”)來記錄、存取及蒐集您的線上行為資料，這有助於我們了解您如何使用我們的服務，以確保能提供您更好的服務體驗。

  3. 資料利用之期間、地區、對象及方式
    1. 利用期間

      自您開始使用本服務開始，至本服務結束或是您提出停止繼續再利用的請求。但法令另有規定或依法需保存之資料者，依其規定。

    2. 利用地區

      樂點股份有限公司所在地區與提供服務地區。此外，為提供本服務，您的個人資料可能被傳輸至本公司關係企業、合作夥伴或服務供應商所在的其他國家或地區，並適用當地的個人資料保護法規。我們將採取適當的安全措施，確保您的個人資料在跨境傳輸過程中及存儲後仍獲得妥善保護。

    3. 利用對象

      我們只會在本政策的目的範圍內使用您的個人資料。除非有下列的情形，我們不會將您的個人資料提供給其它第三人：

      1. 取得您的事前同意；

      2. 法律要求揭露您的特定資訊；

      3. 為提供本服務而有提供給關係企業或合作的第三方之必要；

      4. 當您使用本服務時的行為，違反服務條款或可能損害或妨礙本公司與其他使用者權益或導致任何人遭受損害時，揭露您的個人資料是為了辨識、聯絡或採取法律行動所必要者。

    4. 利用方式
      1. 本公司經營的應用程式或網站

        我們可能使用您的個人資訊，以提供、維護及改善本服務，並防止非法活動或違反使用者條款之行為發生，或於進行會員驗證、活動獎項頒發等情形時，要求您協助提供可證明個人身份之證件、地址、匯款帳戶等資料，以確定使用者為您本人。

      2. 客服意見之處理

        若您向本公司反應您於使用本服務的問題時，我們可能使用您的個人資料以解決相關問題。

      3. 建立及提供統計資訊

        我們會基於統計及分析目的使用您的個人資訊，資訊將會以去識別化之形式使用，亦可能將您的身分、交易資訊提供給我們關係企業或合作夥伴進行調查、分析，以改善本服務的運作、提升產品與服務之效能，並作為研發新產品與服務的參考。

      4. 提供會員服務

        如本公司因您進行交易或退款等，而有提供金流、物流服務必要，本公司有可能會提供您必要的個人識別資料給相關機關或合作、協力廠商作為交易身分驗證或商品、回饋項目送達之用，或與信用卡發卡機構、匯款代收銀行等金融機構，核對付款人、匯款人資訊，以確認為有效付款。

      5. 紅利或回饋金兌換

        本公司如提供紅利或回饋金，您同意委託本公司於兌換之特定目的必要範圍內(含稅務)將您個人資料提供予提供商品或服務之參加廠商、政府機關進行蒐集、處理及利用。

      6. 行銷及優惠推廣服務

        在您加入會員或使用本公司網站或其所附屬網站、網頁或其他服務時， 本公司為提供行銷、與合作夥伴共同行銷或其他服務之目的，將透過行動推播通知、簡訊、寄送電子郵件或其他方式分享優惠消息。若您不希望收到相關優惠或行銷資訊，可隨時取消通知服務。

      7. 我們對於Cookie的利用包括以下幾種類型：

        使用偏好：我們會利用Cookie來記住您的設置和偏好。

        表現：我們使用這些Cookie來收集有關您如何與我們的服務互動的資訊，並幫助我們做相關改善。例如，我們可能會使用這些Cookie來確認您是否與某個服務進行互動。

        安全支付：我們使用這些Cookie來幫助識別和預防安全性的風險。

        分析：我們使用這些Cookie來幫助我們了解和改進我們的服務。例如，我們可能會使用這些Cookie來了解哪些功能最受我們的用戶歡迎，以及我們可以在哪些方面進行改進。

        廣告：我們和我們的廣告合作夥伴使用這些Cookie來投放廣告，使廣告對我們網站的訪問者更相關和更有意義，並跟踪我們的廣告活動在我們的服務和其他網站上的有效性。

  4. 您得行使之權利及方式
    1. 您可以得透過本公司客服信箱(B2CCS@gashpoint.com)請求行使以下權利：

      1. 查詢或請求閱覽。

      2. 請求製給複製本。

      3. 請求補充或更正。

      4. 請求停止蒐集、處理或利用。

      5. 請求刪除。

    2. 權利限制

      若您的請求不符合申請程序或法律另有規定者，前項所列權利將會受到限制。

    3. 酌收費用

      若您依本條第一項向本公司查詢、請求閱覽或製給複製本，本公司將酌收必要成本費用。

    4. 權利行使之方式

      本條第一項權利之行使，應填具申請文件，並向本公司提出可資確認之身分證明文件。委託他人代為申請者，應出具委任書並提供本人及代理人之身分證明文件。

  5. 第三方服務整合

    為了提供您更完善的服務體驗，本公司可能會在本服務內或透過本服務提供來自第三方之軟體應用程式供您選擇是否使用，這些軟體應用程式可能會個別蒐集、處理及使用您的相關識別資訊，例如電話號碼、帳號名稱、個人檔案及登入驗證等。如果這些第三方服務需要您的資訊才能進行，我們可能會提供您必要的個人資訊予該第三方服務。

    另外提醒您，本公司並不擁有或控制這些第三方服務網站或應用程式，其服務皆有個別獨立之隱私權政策及安全措施，建議您於使用該第三方服務前，請詳閱其官方網站或應用程式內所公布之條款、隱私權政策及(或)任何其他聲明，以確認是否繼續使用該第三方所提供之服務。

  6. 您不提供完整之個人資料時可能對您權益造成的影響

    若您無法提供完整且確實之個人資料，您可能會無法順利使用本服務，因此建議您提供完整且確實之個人資料，並應隨時更新。

    若您不願接受Cookie的寫入，您可在瀏覽器功能中設定隱私權等級為高，但可能會導致本服務的部分功能無法正常執行。

  7. 個人資料之安全維護

    本服務均設有防火牆、防毒系統等相關的各項資訊安全設備及必要的安全防護措施以保護您的個人資料，如因業務需要有必要委託其他單位提供服務時，我們也會嚴格要求其遵守保密義務，採取必要檢查程序並確實遵守。

  8. 隱私權政策的修改

    本政策可能會為了要因應社會環境及法令規章的變更或科技的進步而修正，我們會將相關的修正於本服務內公告，請隨時留意查閱本政策的更新資訊。本政策之修正於公告時生效，當您於本政策修正後繼續使用本服務，將表示您已閱讀、瞭解並同意接受本政策的修正。
